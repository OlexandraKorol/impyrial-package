# Impyrial Package

This project demonstrates creation of a custom Python package and usage of relative and absolute imports.

## Structure

impyrial/
- utils.py
- example_script.py
- length/
  - core.py
  - api.py

## Features

- Unit conversion (inches, feet, yards, meters)
- Demonstration of relative imports
- Demonstration of absolute imports

## How to run

Run as module:

python -m impyrial.example_script

## Key Concepts

- Relative imports (. and ..)
- Absolute imports
- Package structure
- Import errors and their causes
